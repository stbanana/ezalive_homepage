# 设计文档

## 项目目标
- 静态部署的公司门户站点（Next.js + Fumadocs + MDX + TailwindCSS）
- 多语言（zh/en）内容驱动
- 首页/产品中心/SEO/表单集成

## 受众与场景
- 访客：了解公司与产品、提交咨询
- 管理员：更新内容、发布版本

## 信息架构（当前）
- /（重定向到 /zh）
- /zh
- /en
- /zh/products
- /en/products
- /zh/products/ez40004
- /en/products/ez40004
- /zh/about
- /en/about
- /zh/contact
- /en/contact

## 内容模型（MDX）
- Home：首页内容（`content/{locale}/home.mdx`）
- Product：产品详情（`content/{locale}/products/*.mdx`）

## 关键页面要点
- 首页：价值主张、优势、CTA、联系表单区块
- 产品中心：列表 + 详情（下载/说明书放在产品页）
- 关于我们：公司介绍内容
- 联系：独立页面，展示联系方式

## 技术与约束
- Next.js 15 App Router + 静态导出（next export）
- Fumadocs UI + Fumadocs MDX
- TailwindCSS v4（Fumadocs 主题样式）
- 顶部导航“产品中心”采用 type: 'custom' + ProductMenuNav 组件，支持悬停弹出分组与卡片，点击可跳转产品中心页
- package.json scripts 配置支持 GitHub Pages + CNAME 部署
- 目标部署平台：Vercel/Netlify/GitHub Pages
- AI 交接文件：AI/HANDOFF.md、AI/TODO.md、AI/SESSION_LOG.md

## 未决事项
- 设计语言与品牌规范（暂无 Logo）
- 真实内容与媒体资源
- 表单集成渠道（如第三方表单服务/自建 API）
- SEO 关键词与元信息
- 新增产品时补充内容与菜单分组
- 视觉风格持续优化
- 适配更多终端与无障碍体验
- 丰富 SEO 元信息与内容
